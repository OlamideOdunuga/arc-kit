How it works
Three prompts. One project.

Answer two questions. Get a full Next.js project with App Kit wired up and testnet config included.

arc-appkit
$ npx arc-appkit create 🔵 Welcome to Arc App Kit Generator   ? What is your project name? my-arc-app ? Which capability? Unified Balance   ✅ Project created successfully! → package.json · .env.example · src/app.js   $ cd my-arc-app && npm install && npm run dev
