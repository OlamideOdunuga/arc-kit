Bridge

Transfer USDC from Ethereum Sepolia to Arc Testnet using Circle's CCTP bridging. The generated project handles the full cross-chain flow server-side — no frontend wallet complexity.

  • app/page.js — bridge UI
  • app/api/bridge/route.js — SDK call
  • .env.example — pre-filled RPC config
  • README.md — setup steps
import { AppKit } from '@circle-fin/app-kit' // in your API route const result = await kit.bridge({ from: { chain: 'ETH-SEPOLIA' }, to: { chain: 'ARC' }, amount: '1.00', token: 'USDC', })

Swap

Exchange USDC for EURC on Arc Testnet in a single SDK call. The generated project includes an amount input, a swap button, and a server-side API route that executes the swap.

  • app/page.js — swap UI with amount input
  • app/api/swap/route.js — SDK call
  • .env.example — pre-filled RPC config
  • README.md — setup steps
import { AppKit } from '@circle-fin/app-kit' // in your API route const result = await kit.swap({ chain: 'ARC', fromToken: 'USDC', toToken: 'EURC', amount, })
